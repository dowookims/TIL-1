# Django 표준 CRUD와 RESTful API

## RESTful API

- **REST** : **Re**presentational **S**tate **T**ransfer

## 1. Read

- `articles/` : 리스트 페이지
- `articles/1` : 디테일 페이지

## 2. Create

- `articles/new` : 글 작성 페이지
- `articles/create` : 글 작성 DB 반영

## 3. Update

- `articles/1/edit` : 글 수정 페이지
- `articles/1/update` : 글 수정 DB 반영

## 4. Delete

- `articles/1/delete` : 글 삭제 페이지
